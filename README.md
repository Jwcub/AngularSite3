# Laboration 4, Angular 2
Denna applikation är skapad i kursen Programmering i TypeScript vid MittUniversitetet. Applikationen är utvecklad i Angular med TypeScript.

## Funktioner
* Kurser läses in från Mittuniversitetet med hjälp av Api och services i Angular.
* Kurser kan sorteras i fallande och stigande ordning vid tryck på tabellhuvud.
* Kurser kan filteras med sökfras
* Sidan uppdateras dynamiskt utan sidomladdning

## Tekniker
* Angular
* TypeScript
* HTML & CSS

## Kom igång
1. Installera beroenden:
`` npm install ``

2. Starta utvecklingsservern:
`` ng serve ``

3. Öppna webbläsaren och gå till:
`` http://localhost:4200 ``
